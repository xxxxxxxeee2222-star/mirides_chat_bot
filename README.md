# TelegramMiridesBot

Очень простой бот.

Пиши только так:

```text
nick ваш_ник
online
chat ваш_текст
```

Примеры:

```text
nick Kuni_Masterr
online
chat всем привет
```

Что делает:
- `nick` сохраняет ваш игровой ник;
- `online` показывает онлайн сервера;
- `chat` отправляет сообщение в Minecraft-чат от вашего ника.
- после `chat` бот оставляет в Telegram запись вида `ваш_ник: сообщение`.

Бот также пытается удалить сообщение пользователя после обработки.
